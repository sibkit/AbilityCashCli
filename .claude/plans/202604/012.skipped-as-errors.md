# 012 — Skipped files → errors

## Проблема

Сейчас файлы без importer'а и пустые результаты помечаются как `skipped` — не вызывают rollback и печатаются обычным цветом. Хотим: любой не-OK файл → ошибка с общим rollback и красным выделением.

## Файлы

- `AbilityCashCli/src/Import/BulkImportRunner.cs` — убрать список `skipped` и обе ветки `_report.Skip(...)`; оба случая оборачивать в `ImportError` и добавлять в `erroredFiles` через `_report.FileErrors(...)`. `Summary` без поля `Skipped`.
- `AbilityCashCli/src/Cli/ImportReportWriter.cs` — удалить метод `Skip`.

## Шаги

1. В `BulkImportRunner.RunAsync`: ветка «нет importer'а» → `FileErrors` с одним `ImportError(source, null, "route", "нет importer'а")`, `erroredFiles.Add`.
2. Ветка `RowsRead == 0` → `FileErrors` с `ImportError(source, null, "import", "пусто")`, `erroredFiles.Add`.
3. Удалить `skipped` список; убрать его из блока summary и из `Summary` record.
4. `ImportReportWriter.Skip` удалить.
