# 017 — Recurrence = "{}" вместо NULL

## Проблема

`TransactionGroups.Recurrence` у старых записей — TEXT `"{}"`, у наших импортированных — NULL. AbilityCash падает при открытии БД. DataGrip ловит `InvalidNumber` при ручном вводе в ячейку (affinity NONE + его UI-редактор), но SQL `UPDATE` пишет `'{}'` нормально.

## Файлы

- `AbilityCashCli/src/Import/AbilityCashValues.cs` — добавить `RecurrenceEmpty = "{}"`.
- `AbilityCashCli/src/Import/TransactionGroupAllocator.cs` — проставлять `Recurrence = AbilityCashValues.RecurrenceEmpty` в `NewGroupAsync`.
- `CLAUDE.md` — в разделе про `TransactionGroup` дописать: `Recurrence` всегда `"{}"`, NULL валит AbilityCash.

## Шаги

1. Константа `RecurrenceEmpty = "{}"` в `AbilityCashValues`.
2. В `TransactionGroupAllocator.NewGroupAsync` выставить `Recurrence` при создании.
3. Обновить CLAUDE.md.
4. Разовая миграция текущей БД: `UPDATE TransactionGroups SET Recurrence='{}' WHERE Recurrence IS NULL` — выполнит пользователь перегонкой импорта или руками.
