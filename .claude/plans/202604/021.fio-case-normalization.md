# 021 — Нормализация регистра ФИО

## Проблема

`PersonNameNormalizer.Normalize` не приводит регистр. Альфа-реестр отдаёт ФИО строкой `"АБДУРАСУЛОВА М.Х."` — на выходе UPPERCASE, не совпадает с существующим счётом `[З] Абдурасулова М.Х.` → ошибка резолва.

## Файлы

- `AbilityCashCli/src/Import/PersonNameNormalizer.cs` — добавить case-нормализацию.

## Шаги

1. Хелпер `TitleCase(word)`: первая буква каждой группы букв (разделители: не-буквы, включая `-`) — `ToUpperInvariant`, остальные — `ToLowerInvariant`. Покрывает `АБДУРАСУЛОВА` → `Абдурасулова`, `СИДОРОВА-ИВАНОВА` → `Сидорова-Иванова`.
2. Хелпер `UpperLetters(word)`: все буквы — `ToUpperInvariant` (для токенов-инициалов с точкой: `м.х.` → `М.Х.`).
3. В ветке 3+ токенов — фамилию (склейку первых N-2 токенов) пропустить через `TitleCase`; `ToInitial` уже даёт точку, букву поднять через `ToUpperInvariant`.
4. В ветке 1–2 токенов — каждый токен: содержит `.` → `UpperLetters`, иначе → `TitleCase`.
5. Алиасы — без изменений: lookup после нормализации.

## Риски

Существующие алиасы в `config.json` уже в title-case (`Ёлкин И.С.`, `Кузьменко С.Р.`, `Пичугин А.П.`) — lookup продолжит срабатывать.
